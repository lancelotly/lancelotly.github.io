Lancelot·Sirius·lyhttps://lancelotly.ml/2017-05-06T18:00:00+08:00Using Dislocker on Ubuntu to Mount BitLocker Disks - 在 Ubuntu 上怎样用 Dislocker 去使用 BitLocker 加密的U盘和硬盘2017-05-06T18:00:00+08:002017-05-06T18:00:00+08:00lancelotlytag:lancelotly.ml,2017-05-06:/2017/Sat 06 May 2017 06:00:00 PM MST/<p>After I encrypted my USB stick with <a href="https://docs.microsoft.com/en-us/windows/device-security/bitlocker/bitlocker-overview">Microsoft's BitLocker</a>, I found that I need to find a solution to make it work in my Ubuntu system either. The best way to solve this problem so far is using <a href="https://github.com/Aorimn/dislocker">Dislocker</a> by Aorimn. </p> <p>As said in the description:</p> <blockquote> <p>This software has been designed to read BitLocker encrypted partitions under a Linux system.</p> <p>The driver has the capability to read/write on:</p> <ul> <li>Windows Vista, 7, 8, 8.1 and 10 encrypted partitions - that's AES-CBC, AES-XTS, 128 or 256 bits, with or without the Elephant diffuser, encrypted partitions;</li> </ul> </blockquote> <h2>Install</h2> <p>It is easy to install Dislocker on Ubuntu.</p> <p>Get the source code: (or unzip one from the release <a href="https://github.com/Aorimn/dislocker/releases">https://github.com/Aorimn/dislocker/releases</a>.)</p> <p><code>git clone https://github.com/Aorimn/dislocker.git</code></p> <p>Since I am on Ubuntu 17.04 now, I need the dependencies include:</p> <p><code>sudo apt install gcc cmake make libfuse-dev libmbedtls-dev ruby-dev</code></p> <p>Then just follow the <a href="https://github.com/Aorimn/dislocker/blob/master/INSTALL.md">INSTALL.md</a> for making and installing.</p> <div class="highlight"><pre><span></span>cmake . make sudo make install </pre></div> <p>To uninstall:</p> <p><code>sudo make uninstall</code></p> <p>We will have the latest version in this way. Otherwise, we can also install one (an old version) from Ubuntu Software Center.</p> <h2>Mounting</h2> <p>To mount a BitLocker drive we need two folders which are used as Dislocker container and drive files mount point.</p> <p>So we need to have two folders (either on /media or /home; and the names we like. here I use "dislocker-container" and "drive" at /media as an example):</p> <div class="highlight"><pre><span></span>sudo mkdir /media/dislocker-container sudo mkdir /media/drive </pre></div> <p>After this preparation, we need to get the path of the drive:</p> <p><code>sudo fdisk -l</code></p> <p>we keep the drive path, the "x" here represent the drive we need: </p> <blockquote> <p>/dev/sdx:....</p> </blockquote> <p>Then we mount the dislocker-contaner:</p> <p><code>dislocker-fuse -v -V /dev/sdx -u -- /media/dislocker-container</code></p> <p>-u here let us type in the BitLocker password.</p> <p>Then we mount files:</p> <p><code>sudo mount -o loop /media/dislocker-container/dislocker-file /media/drive</code></p> <p>Now we can access the BitLocker drive at /media/drive.</p> <p>To unmount the drive:</p> <p><code>sudo umount drive &amp;&amp; sudo umount dislocker-container</code></p> <p>I faced that I could not write to the drive in GNU Files but not through Terminal at the beginning. It was self-fixed after reboot.</p> <h2>Shell file</h2> <p>We have to run such two lines every time. It seems not good. So, I write a .sh shell script for mounting with a single run. </p> <p>First, we need to find the UUID (or part-UUID) of the USB stick to get rid of searching for a changeable drive name in /dev/sdx. We use "blkid" command:</p> <p><code>sudo blkid</code></p> <p>I have my USB stick:</p> <blockquote> <p>/dev/sdb1: PARTUUID="00000000-00"</p> </blockquote> <p>Then we write the .sh script:</p> <div class="highlight"><pre><span></span> <span class="c1">#!/bin/bash</span> <span class="nb">echo</span> <span class="s2">&quot;Preparing for mounting disk...&quot;</span> sudo dislocker-fuse -v -V /dev/disk/by-partuuid/00000000-00 -u -- /media/dislocker-container <span class="nb">echo</span> <span class="s2">&quot;Loading....&quot;</span> sudo mount -o loop /media/dislocker-container/dislocker-file drive <span class="nb">echo</span> <span class="s2">&quot;Done!&quot;</span> </pre></div> <p>That's it! We just type (or double click to choose run-in-terminal):</p> <p><code>./start.sh</code></p> <p>Also, the un-mount shell script:</p> <div class="highlight"><pre><span></span> <span class="c1">#!/bin/bash</span> <span class="nb">echo</span> <span class="s2">&quot;Preparing for unmounting disk...&quot;</span> sudo umount drive <span class="o">&amp;&amp;</span> sudo umount dislocker-container <span class="nb">echo</span> <span class="s2">&quot;Done!&quot;</span> </pre></div> <p>&#x1F411;</p>推荐 TheFatRat 的
